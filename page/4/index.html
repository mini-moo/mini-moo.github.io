vim编辑器使用

发表于 2016-04-11   |  
vim，一个无比性感的编辑器；
vim的6种模式：
1.普通模式：可使用编辑器命令；
2.插入模式：可插入文本；
3.可视模式：与普通模式类似，但移动命令会扩大高亮的文本区域；
4.选择模式：可用鼠标或光标高亮文本；
5.命令行模式：输入会被解释或命令的文本；
6.Ex模式：与命令行模式类似，在使用：visual命令离开Ex模式前，可以执行多余命令.

ubuntu14.04 lts 下安装chrome

发表于 2016-03-15   |  
ubuntu14.04默认安装的是firefox浏览器，但喜欢chrome所以果断安装了！



64位下终端命令：

wget https://dl.google.com/linux/direct/google-chrome-stable_current_amd64.deb

然后再在终端输入：

 sudo dpkg -i google-chrome-stable_current_amd64.deb

此时如果发现安装的错误，其实是lib没有配置：

sudo apt-get install -f

然后输入前面的命令：

 sudo dpkg -i google-chrome-stable_current_amd64.deb

然后在usr/share/applications中可以找到chrome图标双击即可使用（你可以固定到启动器中）
