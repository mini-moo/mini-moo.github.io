c++虚函数

发表于 2016-05-07   |  
 c++在运行时的多态性主要是通过虚函数来实现的。虚函数是重载的另一种表现形式，只是一种动态的重载方式，它提供了一种灵活的，运行时的多态性机制。



#include<iostream>
using namespace std;
class Base {
public:
    Base(int x, int y)
    { a = x; b = y;}
    void show()
    {
        cout << “Base:”;
        cout << a << “ “ << b << endl;
    }
private:
    int a, b;
};
class Derived:public Base {
public:
    Derived(int x, int y, int z):Base(x, y)
    { c = z; }
    void show()
    {
        cout << “Derived:”;
        cout << “c=” << c << endl;
    }
private:
    int c;
};
int main(void)
{
    Base mb(60, 60), pc;
    Derived mc(10,20,30);
    pc = &mb;
    pc -> show();
    pc = &mc;
    pc -> show();
    return 0;
}
程序运行结果如下：


Base:60 60

Base:10 20


不是预想的：



Base:60 60

Derived:30


这令我很是尴尬，查看原因：

在c++中规定，基类（Base）的对象指针（pc）可以指向它公有派生（Derived）的对象（mb），但是当有指向公有派生类对象（mc）时，他只能访问派生类中从基类继承来的成员，而不能访问公有派生类中定义的成员。所以不管pc指向那个对象，“pc->show()”调用的都是基类定义的show()函数。

解决此问题的方法可以：

1.强制类型转换：


((Derived )pc)->show();


2.使用虚函数：


class Base {
public:
    Base(int x, int y)
    { a = x; b = y;}
    virtual void show()
    {
        cout << “Base:”;
        cout << a << “ “ << b << endl;
    }
private:
    int a, b;
};


当虚函数在派生类中重新定义式，其函数原型，包括：返回类型，函数名，参数个数，参数类型的顺序，都必须与基类的原型完全相同。（友元函数，静态成员函数，内联函数，构造函数不能为虚函数，虚函数必须为其所在类的成员函数）
