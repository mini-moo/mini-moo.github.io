c++中关键字const

发表于 2016-05-16   |  
const(对于单纯的常量比起#define，尽可能使用const)：
const允许指定一个语义约束，即：指定一个不该被改动的对象。它可以在classes外部修饰作用域为全局（global）或命名空间（namespace）中的常量，亦可以修饰文件，函数或区块作用域中被声明为static的对象，同样可以修饰classes中static和non-static成员变量。
对于指针：

char g[] = “hello”;

char k[] = “world”;

char p = g;            

//此时p是自由的，可以指向g，也可以指向k

const char p = g;        

//此时p的值是固定的,如果将g值改成”hi”,则出现错误：

//error: incompatible types in assignment of ‘const char [3]’ to ‘char [6]’

char const p = g;        

//此时p只能指向g，g的值改变时，p也可以改变，如果此时再指向k则会出现错误：

//error: assignment of read-only variable ‘p‘

const char const p = g;    

//此时p只能指向g，且它的值不能改变
如果const出现在星号左边，表示被指物是常量；如果出现在星号的右边，表示指针自身是常量；如果出现在星号的两边，表示被指物和指针都是常量。


const char p;

char const p;

//两种写法的意义一样；
const成员函数：
const可以和函数返回值、各参数、函数自身（成员函数）产生关联，使函数返回一个常量值，往往可以降低因客户错误而造成的影响，又保证安全性和高效性。
const实施于成员函数：
1.可以使class接口容易被理解，可以很容易知道哪个函数可以改动对象内容哪个函数不可以。
2.可操作const对象
注意：成员函数只有在不更改对象的任何成员变量（static成员变量除外）时才可以称为const。
