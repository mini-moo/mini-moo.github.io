Sublime Text 3安装Package Control

发表于 2016-05-02   |  


安装**Package Control（**工欲善其事必先利其器）：
Package Control是一个用于管理插件的好工具，可以用于安装、删除、禁用相应的插件，常用的插件都能在上面找到。其源码地址在https://github.com/wbond/package_control_channel上，安装非常方便，使用git（说到git大家可以学习学习，可以去看《progit》这本书https://github.com/git/git这个网址有时进不去有时进的去，可以翻墙,详情：ss代理
）

简单的办法就是ctrl+` 打开控制台，然后将下面的代码放进去回车：


import urllib.request,os; pf = ‘Package Control.sublime-package’;

ipp = sublime.installed_packages_path();

urllib.request.install_opener( urllib.request.build_opener(urllib.request.ProxyHandler()) ); 

open(os.path.join(ipp, pf),‘wb’).write(urllib.request.urlopen( ‘http://sublime.wbond.net/‘ +pf.replace(‘ ‘,‘%20’)).read())


当插件控制安装好以后，你就可以安装自己喜欢的插件（快捷键：ctrl+shift+p 打开Package Control），装完之后需要进行重启！！！
