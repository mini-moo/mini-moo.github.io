ubuntu14.04 lts 下安装chrome

发表于 2016-03-15   |  
ubuntu14.04默认安装的是firefox浏览器，但喜欢chrome所以果断安装了！



64位下终端命令：

wget https://dl.google.com/linux/direct/google-chrome-stable_current_amd64.deb

然后再在终端输入：

 sudo dpkg -i google-chrome-stable_current_amd64.deb

此时如果发现安装的错误，其实是lib没有配置：

sudo apt-get install -f

然后输入前面的命令：

 sudo dpkg -i google-chrome-stable_current_amd64.deb

然后在usr/share/applications中可以找到chrome图标双击即可使用（你可以固定到启动器中）



32位同样：

 wget https://dl.google.com/linux/direct/google-chrome-stable_current_i386.deb

然后再输入：

sudo dpkg -i google-chrome-stable_current_i386.deb

后面的类似，省略一万字。。。

（chrome自带flash，安装了省下不少时间）

最新版 49的chrome已经发布，ubuntu14.04迟迟未更新

昨日（16.5.4）无聊想将ubuntu14.04升级成16.04，需要将所有软件更新才能更新系统，更新时发现只有chrome不能更新，试了很多网上的方法都不行，所以去谷歌官网下了最新版的chrome（deb包），顿时觉得自己好傻，如此简单，还弄的那么麻烦！安装完chrome却发现只能往15.04升级，顿时哭晕在厕所。探索升级ing…….
